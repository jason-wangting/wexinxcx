# 商品管理

## 1. 文档目标

定义码头版商品主数据边界，支撑销售记录按商品维度标准化。

---

## 2. 当前范围

- 商品档案定位
- 商品基础字段
- 商品与销售明细关系

---

## 3. 当前已确认规则

1. 码头版店铺需要维护商品档案
2. 商品至少包含名称与单价
3. 销售明细优先引用商品档案，必要时保留名称快照
4. 商品停用后不影响历史订单展示

---

## 4. 设计原则

- 商品主数据服务于销售与对账，不独立脱离业务场景
- 商品变更应可追溯（价格、状态）
- 商品查询必须在店铺作用域内执行

---

## 5. 后续待细化

- 规格、单位、分类字段
- 航次临时调价规则
- 港口推荐商品到店铺商品的初始化策略
