# 安全审计专题文档

本目录包含系统安全审计与操作留痕体系的完整设计，适用于微信小程序 + 微信云开发（云函数 + 云数据库）。

## 文档导航

- [操作留痕设计](./操作留痕设计.md) — 总体原则、架构、数据模型、防篡改机制
- [云函数接口规范](./云函数接口规范.md) — 写入协议、接口清单、幂等与鉴权规范
- [数据库与索引清单](./数据库与索引清单.md) — 集合结构、字段约束、索引方案、归档策略
- [动作编码字典](./动作编码字典.md) — action_name 强枚举字典与 action_type 对应关系

## 核心原则

**无审计不写入** — 任何一次 edit 必须同事务写入 `audit_events`，审计失败则业务写入回滚。
