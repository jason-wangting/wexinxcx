0. 渔港交易数字化平台（微信小程序 + 云开发）
1. 系统角色：超级管理员 / 推广员 / 普通用户
2. 产品版本：管理版 / 码头版 / 普通用户版
3. 首次进入先询问是否为码头店铺工作人员
4. 是码头店铺工作人员进入码头版，否则进入普通用户版
5. 所有用户默认普通用户，推广员由超级管理员分配
6. 推广员入口在“我的”页面按角色动态展示
7. 港口由超级管理员维护，店铺必须归属港口
8. 港口一旦选中不可修改
9. 码头版领域主链路：港口 -> 店铺 -> 船只 -> 航次 -> 按人销售订单 -> 按商品销售明细
10. 店铺采用简化 RBAC：管理员 / 成员
11. 记账与销售录入在码头版内按店铺上下文执行
12. 普通用户版展示与当前用户相关的订单与付款信息
13. 按人销售订单承载整单应收、已付、欠款与付款状态
14. 按商品销售明细承载商品行（名称、单价、数量、金额）
15. 支付支持全额、部分、分次与线下核销，状态以后端回调为准
16. 离线场景先本地写入，再通过同步队列回传云端
17. 港口维度可做品类统计与经营大屏，沉淀数据价值
